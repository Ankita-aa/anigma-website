# Frontend Generate Page

Minimal static page for public MVP generation.

## Features

- video upload
- instruction textbox
- generate button
- loading indicator
- error display
- result video player

## Use

Serve this folder with any static server and ensure backend API is running.

Example:

```bash
cd frontend_generate_page
python -m http.server 5173
```

Then open:

`http://127.0.0.1:5173`

By default it calls backend at `http://127.0.0.1:8080`.
